This section contains example project configurations that you can use to get started with Tableau. These examples are meant to be used as a reference and are not meant to be copied directly into your project. They are meant to be used as a reference to help you understand how to configure your project.

:::danger
These examples are not meant to be copied directly into your project. They are meant to be used as a reference to help you understand how to configure your project.
:::

:::note
We are open to collecting more examples from the community. If you have a project configuration that you would like to share, please open a pull request on the [Tableau GitHub repository](https://github.com/ldtteam/Tableau)
:::

:::warning
These examples are not complete, and for example miss the setup in `settings` and `repositories` blocks. These are meant to be used as a reference and not as a complete project configuration.
Follow the [Getting started](../02-getting-started.mdx) guide to get a complete project configuration.
:::
